# github-final-project# Simple Interest Calculator

## Description
This project is a simple interest calculator that computes the interest earned on a principal amount over a given period of time at a specified interest rate.

## Purpose
The purpose of this calculator is to help users easily calculate simple interest for loans or investments.

## Formula Used

Simple Interest = Principal × Rate × Time

Where:
- Principal (P) = initial amount of money
- Rate (R) = annual interest rate (in decimal form)
- Time (T) = time in years

## How It Works
The user provides:
- Principal amount
- Interest rate
- Time period

The calculator multiplies these three values to compute the simple interest.

## Example

If:
- Principal = 1000
- Rate = 5% (0.05)
- Time = 3 years

Interest = 1000 × 0.05 × 3  
Interest = 150

The simple interest earned is 150.
